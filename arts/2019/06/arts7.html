# Algorithm

  • Palindrome Number

8ms | 68.82% Run time
35.1MB | 71.77% Memory

public boolean isPalindrome(int x) {
    boolean isPalindrome = true;
    String strX = Integer.toString(x);

    for (int i = 0; i < strX.length() / 2; i++) {
        if (strX.charAt(i) != strX.charAt(strX.length() - 1 - i)) {
            isPalindrome = false;
            break;
        }
    }

    return isPalindrome;
}
